CO2PPMBadge

Show you care for this planet: easily add an image to any website showing current and past levels of CO2 Parts Per Million (PPM) in the atmosphere

Why an image?

APIs are for nerds, badges and banners are easy to embed on any webpage and are cool!

Banners & Badges

Square banner

shows all the values in a square image in PNG format

Horizontal banner

shows all the values in a horizontal image in PNG format

PPM 00

is the latest available badge

PPM 01

shows the value 1 year before the latest available badge

PPM 10

shows the value 10 years before the latest available badge

How to embed

Right click on any image and select "copy image address" from the contextual menu, the URL can then be used in any HTML or Markdown page.
